# Paraísos Perdidos — MVP para empezar a publicar

Primera aplicación Next.js de Paraísos Perdidos.

## Ejecutar
```bash
npm install
npm run dev
```
Abre http://localhost:3000

## Publicar ahora
Edita `data/works.ts` y añade tus textos. La página `/publicar` permite escribir y previsualizar. El siguiente paso será conectarla a base de datos para publicar sin tocar código.
